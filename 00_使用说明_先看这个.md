# 授课型硕士留学素材包｜使用说明（v3｜2026-02-18）

## 1. 账号定位（运营不得改动）
- 人设：授课型硕士方向留学博主，强调信息密度、可执行性、反焦虑。
- 受众：本科在读、毕业1-3年、预算20-40万、希望提升就业竞争力的人群。
- 内容目标：用户读完就知道下一步动作。

## 2. 分工边界（必须遵守）
- 你（博主）负责：结论、逻辑、案例、反方回应、行动步骤。
- 运营负责：检索信息、按证据卡回填、平台裁剪发布。
- 运营禁止：新增观点、扩写推理、替换核心结论。

## 3. 交付结构（本次已改为双版本）
- `01_首批素材/*.md`：发布母稿（给知乎/公众号直接改编）。
- `01_首批素材/证据回填卡/*.md`：证据卡（运营只回填与复核）。
- `01_首批素材/00_案例索引_首批6篇.md`：案例键总索引。
- `01_首批素材/00_总控面板_首批6篇.md`：主题进度与复核看板。

## 4. 案例键体系（已替代旧 case_id 单键）
- 不再使用“仅 case_id”检索。
- 统一主键：
  - `case_key = case_id|学生姓名|录取学校|录取专业`
  - `case_uid = case_key`（当前版本直接等同，后续可单独编号）
- 检索顺序：先全量匹配 `case_key`，再回看 `case_id`。

## 5. 证据卡字段（统一口径）
每条证据必须包含：
- 数据项
- 结论句
- 数值/规则
- 适用国家/城市
- 口径（是否含学费/是否季调/是否含住宿等）
- 发布/更新日期
- 抓取日期
- 证据状态（已生效 / 已公布待生效 / 待确认）
- 可信度（A官方原始 / B官方二次 / C平台样本）
- 下次复核日期
- 来源链接
- 检索式
- 验收格式
- 填入段落ID
- 备注

## 6. 信息源优先级
1. 学校官网（programme / tuition / entry / deadline / career）
2. 政府与签证官网（GOV.UK / IRCC / Home Affairs / MOM / ImmD）
3. 官方考试机构（IELTS / TOEFL / PTE）
4. 官方统计机构（ONS / ABS 等）
5. 招聘平台（LinkedIn / Indeed，仅做岗位样本）
6. 社媒平台（小红书 / 知乎，仅做痛点与反方样本）

## 7. 时效 SLA
- 签证/工时/毕业工签：30天复核
- 学费/录取要求/DDL：90天复核
- 生活费/住宿费：180天复核
- 宏观就业统计：按官方最新发布周期复核

## 8. 平台不可直取数据的留空规范
遇到无法稳定抓取的数据，不得猜测，必须填写：
- 需要什么信息（字段）
- 如何搜索（关键词 + 时间窗 + 去重）
- 从哪里搜索（平台与入口）
- 验收标准（样本量、输出格式、截图要求）

统一口径建议：
- 时间窗：近90天
- 样本量：不少于50条
- 输出：前20高频词 + 截图 + 去重规则

## 9. 发布前检查清单
- [ ] 母稿无 `{{Dxx}}` 占位符
- [ ] 母稿无绝对路径
- [ ] 所有关键数字在证据卡可追溯
- [ ] 每条证据有状态、可信度、复核日期
- [ ] 不确定信息标记为“待确认”且附检索方案

## 10. 版本与命名
- 母稿：`Sxx_主题_发布母稿_v3_2026-02-18.md`
- 证据卡：`Sxx_主题_证据回填卡_v3_2026-02-18.md`
- 总控：`00_总控面板_首批6篇.md`
