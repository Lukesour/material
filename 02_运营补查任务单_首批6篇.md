# 运营补查任务单（首批6篇｜v2）

> 目标：运营只做信息检索与回填，不改结论，不扩写。  
> 规则：每篇必须回填`D01-D08`，并提交证据回填卡。

## 全局默认参数（如未另写，按此执行）
- 国家包：英国 / 澳洲 / 新加坡 / 中国香港 / 加拿大
- 学位类型：授课型硕士（Taught Master）
- 人群：预算20-40万、希望提升就业竞争力
- 证据时效：签证30天、学费90天、生活费180天

## 平台内难抓数据的留空规范（小红书/知乎/LinkedIn岗位词频）
当无法稳定抓取时，必须在证据卡备注写：
1. 需要什么信息：例如“目标岗位JD高频词（近90天，国家=英国，岗位=Business Analyst）”。
2. 如何搜索：关键词 + 时间过滤 + 地点过滤 + 去重规则。
3. 从哪里搜索：LinkedIn Jobs / Indeed / BOSS直聘（回国岗）/ 小红书站内关键词。
4. 如何验收：至少50条岗位样本，给出前20高频词与截图。

## 回填规范（不变）
- 所有数据必须标注：来源链接 + 发布/更新日期 + 抓取日期 + 口径。
- 冲突时以学校官网/政府官网为准。
- 不确定信息统一标记`[待二次确认]`，不得自行推断。
